Swag Labs Automation Project - UI & API

Automating e-commerce website functionality with Selenium and Python (pytest).

Project Overview 🔍

This framework automates both functional/UI testing and API testing for the SauceDemo e-commerce website using Selenium and Python.

The project includes tests for the website’s flow, covering both happy paths and edge cases, including negative test cases, all executed with the pytest framework. The results are visualized using Allure Reports for better insight.

🧩 By following the Page Object Model (POM), the test scripts are modular, reusable, and easy to maintain.

Features ⚙️

  • 🔑 Automated login functionality testing
  • 🔍 Product search and filtering tests
  • 🛒 Checkout functionality tests with multiple product scenarios
  • 🔗 API Testing for backend validation (Products, Users, and Cart endpoints)
  • ✅ Assertions for API response validation (status codes, response times, and data integrity)

📊 Allure Reports

Allure provides a beautiful and detailed report to help you analyze test results.

API Tests 🧑‍💻

In addition to the UI tests, comprehensive API tests have been implemented to validate the reliability and correctness of backend operations. These tests cover a variety of API endpoints and HTTP methods, including:

  • 🔄 GET: Fetching data
  • ➕ POST: Creating resources
  • ✏️ PUT: Updating resources
  • ❌ DELETE: Deleting resources
  • 🔧 PATCH: Partially updating resources

The key areas tested include:

  • 📦 Product API Testing: Verifies product creation, retrieval, updates, and deletions. Ensures required fields (id, title, price) are correctly structured.
  • 👤 User API Testing: Validates user data, ensuring essential fields (id, email, username) and proper account management are in place.
  • 🛒 Cart API Testing: Ensures cart operations like adding/removing products, updating details, and checking contents are functioning correctly.

Technologies Used 🛠️

  • 🐍 Python
  • 🧪 Selenium for automated browser interactions
  • ⚙️ pytest framework for test automation
  • 📈 Allure for reporting and visualizing test results
  • 🛠️ Postman for API testing and validation
  • 🔧 Git & GitHub for version control
